// Quick start

From pip install to running in 10 seconds.

Import what you need, instantiate the class, call the method. There's no setup, no config files, no global state. Just small, friendly classes.

quickstart.py python 
from cozy_kit import Greeting, Timer, TextStudio, CozyUI

# Personalised greetings
greet = Greeting(name="Youssef", nickname="yo")
print(greet.welcome())            # Welcome Youssef!
print(greet.auto_greet())         # Picks based on time + season

# A Pomodoro that just works
timer = Timer()
timer.pomodoro(work_time=25, break_time=5,
                long_break_time=15, show=print)


# Text tricks
fx = TextStudio()
print(fx.to_morse("hi"))            # .... ..
print(fx.corrupt("vibes", mode="bubble"))   # ⓥⓘⓑⓔⓢ

# Terminal UI helpers
ui = CozyUI()
print(ui.cozy_box("done!"))
Module 01 / Greeting

Greeting

A small class that knows your name, the time of day, the season, and whether today happens to be a holiday. Loads quotes and stories from bundled JSON databases, and lets you add your own.

Greeting(name, age, gender, nickname)

Constructor. All parameters are keyword-only and optional.

welcome() → str

Returns a personalised welcome message.

bye(destination) → str

A goodbye message, optionally addressed to where you're heading.

good_morning() · good_afternoon() · good_evening() → str

A time-of-day greeting paired with a random quote from the bundled library.

good_night() → str

Goodnight wishes plus one of five bundled bedtime stories.

auto_greet() → str

Greets based on the current hour and season — winter, spring, summer, and autumn each have their own time bands. Falls through to holiday_greet() first.

motivate() → tuple[str, str]

Returns (title, content) for a random motivational quote.

fun_fact() · random_quote() → str

A random fun fact, or a random pick from any of the four time-of-day greeters.

holiday_greet() → str | None

Returns a wish for today's holiday, or None if today isn't one. Supports New Year's Day, Valentine's Day, Earth Day, International Friendship Day, World Smile Day, and Children's Day.

add_bedtime_story(title, content)

Adds a new bedtime story to the JSON database. Raises InvalidStoryError on empty input.

add_motivation(name, content) · add_fun_fact(name, content)

Append a new motivational quote or fun fact to the bundled databases.

greeting_example.py python 
from cozy_kit import Greeting

g = Greeting(
    name="Youssef",
    nickname="yo",
    age=8,
)

# Time-aware automatic greeting
print(g.auto_greet())
# Good Morning Youssef!
# Or good morning Yo!
# Anyways, here's a quick morning quote.
#
# Rise & Shine
# Every morning is a fresh start...

# Holidays light up automatically
print(g.holiday_greet())
# 🎉 Happy New Year's Day, Youssef!
# A new year means new chances to grow.

# Get a (title, body) pair
title, body = g.motivate()
print(f"{title}\n{body}")
# One Step
# You don't have to see the whole staircase...

# Add your own bedtime story
g.add_bedtime_story(
    title="The Sleepy Robot",
    content="Once upon a circuit board...",
)

# Goodbye with a destination
print(g.bye(destination="the gym"))
# Bye Youssef! Have a nice day at the gym!
Module 02 / Timer

Timer

Countdowns, Pomodoro cycles, stopwatches, and waits — all with a single consistent API. Every blocking timer takes a show callback so you can pipe the output anywhere: stdout, a curses pane, a Tkinter label.

countdown(count, time_type, show)

Counts down from count. time_type is one of "sec", "min", or "hour". Fires a desktop notification when it ends.

pomodoro(work_time, break_time, long_break_time, show)

Runs a full Pomodoro cycle: work → break → work → break → … → long break after every 4 work sessions. Every transition sends a notification with plyer.

pause_pomodoro() · resume_pomodoro() · stop_pomodoro()

Control a running Pomodoro from another thread. Each raises a specific error if called in the wrong state.

wait(count, time_type)

A friendlier time.sleep() — accepts "sec", "min", or "hour".

get_time() → str

Returns the current local time as HH:MM:SS.

start_stopwatch() · end_stopwatch() → timedelta

A simple stopwatch pair. end_stopwatch() returns a datetime.timedelta and resets the state so you can reuse the instance.

timer_example.py python 
from cozy_kit import Timer

t = Timer()

# A 5-minute countdown printed to stdout
t.countdown(count=5, time_type="min", show=print)
# 04:59  04:58  04:57 ...  00:00
# ⏰ Time's up!

# A full Pomodoro session
t.pomodoro(
    work_time=25,
    break_time=5,
    long_break_time=15,
    show=print,
)

# Stopwatch
t.start_stopwatch()
# ... do some work ...
elapsed = t.end_stopwatch()
print(f"Took {elapsed}")
# Took 0:03:12.481923

# Current time, nicely formatted
print(t.get_time())   # 14:32:08

# Friendly sleep
t.wait(count=2, time_type="sec")
Module 03 / TextStudio

TextStudio

A grab-bag of text utilities — the kind you reach for half-way through every script. Every method validates non-empty input and raises a clean EmptyTextError if you forget.

to_morse(text) → str

Encodes letters, digits, and common punctuation into Morse. Unknown characters become ?.

to_upper / to_lower / to_title(text)

Standard case conversions, with empty-input guards.

snake / kebab_case / pascal_case(text)

Convert spaced text to snake_case, kebab-case, or PascalCase.

caesar(text, shift) → str

A Caesar cipher with full negative-shift support. Non-alphabetic characters pass through untouched. Raises InvalidShiftError if shift isn't an int.

reverse(text) · word_count(text) · char_count(text) · count_vowels(text)

The usual little measurements and transforms.

remove_spaces(text) · remove_punctuation(text) · replace_with_spaces(text, replacement)

Strip or substitute whitespace and punctuation.

space_out_letters(text) → str

Spreads letters apart: "hi" → "h i".

corrupt(text, mode) → str

Reskin text in one of four moods: "glitch" (combining diacritics), "broken" (ASCII-art letters), "bubble" (Unicode circled), or "void" (occult-y symbols).

text_studio_example.py python 
from cozy_kit import TextStudio

fx = TextStudio()

# Morse
fx.to_morse("hello world")
# .... . .-.. .-.. --- / .-- --- .-. .-.. -..

# Case transforms
fx.snake("Cozy Kit Demo")        # cozy_kit_demo
fx.kebab_case("Cozy Kit Demo")   # cozy-kit-demo
fx.pascal_case("cozy kit demo")  # CozyKitDemo

# Caesar (shifts roll over, negatives ok)
fx.caesar("hello", shift=3)     # khoor
fx.caesar("khoor", shift=-3)    # hello

# Counts & transforms
fx.word_count("hi there friend")   # 3
fx.count_vowels("cozy kit")         # 2
fx.reverse("warmth")               # htmraw

# Corruption modes
fx.corrupt("vibes", mode="bubble")   # ⓥⓘⓑⓔⓢ
fx.corrupt("hi",    mode="broken")   # #!
fx.corrupt("void",  mode="void")     # V̴⊘ɨ◊
fx.corrupt("hi",    mode="glitch")   # ̴h̶̵i̷
Module 04 / CozyUI

CozyUI

Pretty box-drawing for terminals. Every helper returns a plain string — so you can print(), log, or splice them anywhere. Auto-sizing column widths included.

divider(symbol="═", length=30) → str

A horizontal rule of any character, any length.

cozy_title(text, symbol="═", length=15) → str

Wraps text in equal-length dividers on each side.

cozy_table(headers, rows) → str

A full box-drawn table with auto-computed column widths. headers is a list of strings, rows is a list of equal-length lists.

progress_bar(percent) → str

A 10-block bar — full blocks for completed deciles, dashes for the rest, percentage on the end.

cozy_box(text) → str

A single-line message wrapped in a tidy double-line box.

Details

Details()

Errors

Named errors, honest tracebacks.

Every failure mode gets its own exception class, all inheriting from CozyKitError. Catch the base class to handle anything cozy-kit might raise, or narrow it down for surgical handling.

error_handling.py python 
from cozy_kit import errors, Timer

t = Timer()

try:
    t.end_stopwatch()       # never called start_stopwatch()
except errors.StopWatchNotStartedError as e:
    print(f"Oops: {e}")

try:
    t.countdown(count=5, time_type="decade", show=print)
except errors.InvalidTimeTypeError as e:
    print(f"Use sec / min / hour: {e}")
Exception hierarchy
CozyKitError EmptyTextError InvalidTimeTypeError InvalidTimeAmountError InvalidShiftError InvalidStoryError StopWatchNotStartedError DatabaseNotFoundError PomodoroNotStartedError PomodoroNotPausedError InvalidMotivationNameError InvalidMotivationContentError InvalidFunFactError TimerAlreadyRunningError
Heads up

Always catch CozyKitError if you want a single safety net around any cozy-kit call. Every specific error inherits from it, so nothing slips through.
